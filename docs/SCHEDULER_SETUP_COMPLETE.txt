================================================================================
                        HOW IT WORKS (OVERVIEW)
================================================================================

STEP 1: System Clock
       Every minute at :00 seconds, Linux triggers cron daemon

STEP 2: Cron Daemon
       Reads crontab file for www-data user
       Finds: * * * * * cd /var/www/html/it_ticket_system && php artisan schedule:run
       Executes the command

STEP 3: Laravel Scheduler
       Runs: php artisan schedule:run
       Reads: routes/console.php
       Checks: "Are any tasks due RIGHT NOW?"

STEP 4: Task Execution
       If task is due:
       - Executes: php artisan tickets:send-approval-reminders
       - Queries database for pending tickets
       - Sends emails to approvers
       - Logs result in storage/logs/laravel.log

STEP 5: Return
       Exit and wait for next minute

REPEAT every 60 seconds, 24/7, forever

1. Verify cron is running: sudo systemctl is-active cron
2. Reinstall job if needed: sudo crontab -u www-data -e
3. Enable auto-start: sudo systemctl enable cron
